## M5.1 Requirements (Roadmap Lines 117-147)

### Fix #1: CMakeLists.txt Optimization (CRITICAL, EASY)
**Requirement**: Move `-O2` from CMAKE_CXX_FLAGS_RELEASE to base CMAKE_CXX_FLAGS
**Rationale**: OJ builds without CMAKE_BUILD_TYPE, so -O2 must be unconditional
**Impact**: Without this fix, code runs at 18.6s (FAILS 16s limit) vs 15.2s with optimization

### Fix #2: File Count Violation (CRITICAL, DESIGN CHANGE)
**Requirement**: Eliminate `.tmp` file creation in delete operation
**Rationale**: Delete creates temporary files (e.g., `data_05.bin.tmp`), pushing count to 21 files (exceeds 20-file limit)
**Impact**: OJ will reject with resource limit error

### Fix #3: Insert Performance Optimization (CRITICAL, ARCHITECTURAL)
**Requirement**: Fix O(n^2) insert complexity from full bucket scan on every insert
**Options**:
  a. Remove duplicate check entirely (verify if spec allows)
  b. Optimize duplicate check (sorted bucket + binary search)
  c. Use in-memory bloom filter or hash set for current session
**Impact**:
- Random 100K: 15.2s (borderline)
- Insert-heavy 100K: 16.9s (FAILS limit)
- Same-key 50K: 129s (catastrophic)
