## Binary Format Verification

**Test:** Insert two entries and examine binary format
```
insert test 123
insert test 456
```

**Hexdump:**
```
00000000  04 74 65 73 74 7b 00 00  00 01 04 74 65 73 74 c8  |.test{.....test.|
00000010  01 00 00 01                                       |....|
```

**Format breakdown:**
- `04` = length byte (4 for "test")
- `74 65 73 74` = "test" in ASCII
- `7b 00 00 00` = 123 in little-endian int32
- `01` = flags (active)

✅ **VERIFIED:** Binary format is correct
