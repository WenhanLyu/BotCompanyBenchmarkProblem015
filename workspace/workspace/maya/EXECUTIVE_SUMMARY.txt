MATHEMATICAL PROOF:

  For n inserts to same bucket:
  - Insert #1 checks 0 existing entries
  - Insert #2 checks 1 existing entry
  - Insert #n checks (n-1) existing entries

  Total comparisons: 0 + 1 + 2 + ... + (n-1) = n(n-1)/2 = O(n²)

  With 33,000 inserts and average bucket size of 500:
  - Total comparisons: 33,000 × 500 = 16.5 million
  - Plus string operations → 77s CPU time
